Teknoo Software - ReactPHP Symfony Bridge
=========================================


Installation & Requirements
---------------------------
To install this library with composer, run this command :

    composer require teknoo/reactphp-symfony

This library requires :

    * PHP 7+
    * Composer
    * Symfony 3.2+
    * ReactPHP 0.5+
